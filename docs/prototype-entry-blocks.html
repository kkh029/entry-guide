엔트리 블록 프로토타입 — Phase 2 전체

지원 모양: hat / stack / cap / reporter / boolean / cblock / cblock_double. 지원 필드: text / input / dropdown / icon / block (자식 끼움).

1) hat — 시작 카테고리

2) stack — 일반 명령

3) cap — 마무리

4) reporter — 값 블록

5) boolean — 판단 블록 (양옆 뾰족 육각형)

6) c-block — 흐름 (단일 슬롯)

7) c-block 이중 슬롯 — 만일 ~ 아니면

8) block 필드 — 자식 reporter / boolean 끼우기

사용법

HTML data-block 속성 (JSON)

<script src="assets/entry-blocks/renderer.js" defer></script>

<div class="entry-block" data-block='{
  "shape": "stack",
  "category": "MOVING",
  "contents": [
    { "type": "text",  "value": "x 좌표를" },
    { "type": "block", "block": {
        "shape":"reporter","category":"VARIABLE",
        "contents":[{"type":"text","value":"걷는속도"}]
    }},
    { "type": "text",  "value": "만큼 바꾸기" }
  ]
}'></div>

JS 호출

EntryBlocks.render('#my-block', {
  shape: 'cblock', category: 'FLOW',
  contents: [{type:'input',value:'10'},{type:'text',value:'번 반복하기'}],
  statements: [
    { shape:'stack', category:'MOVING', contents:[{type:'text',value:'x 좌표를'},{type:'input',value:'10'},{type:'text',value:'만큼 바꾸기'}] }
  ],
});
옵션: shape · category · icon(hat 왼쪽) · marker · contents[text|input|dropdown|icon|block] · statements(c-block 자식) · middleContents(이중 슬롯 가운데 띠)
