새 페이지 제작 가이드

최종 갱신: 2026-05-28

개요

이 문서는 docs/에 새 페이지를 추가할 때 따라야 하는 레이아웃·구조·체크리스트를 정리한다. 빌드 툴이 없는 정적 HTML 사이트이므로, 신규 페이지는 기존 파일을 복사·수정하는 방식으로 만든다.

추천 베이스 파일: docs/entry-game-2-velocity.html (CTA·TOC·메타박스 구조가 가장 최신).

1. 파일명 규칙

  • 에피소드: entry-game-N-<slug>.html (예: entry-game-4-score.html)
  • 독립 소개 페이지: about-<slug>.html
  • 슬러그는 영어 소문자 + 하이픈만 사용

2. 전체 파일 뼈대

아래가 신규 페이지의 최소 완성형이다. <!-- FILL --> 부분만 채우면 된다.

<!-- 사이드바 목차 스크립트: section.card h2를 자동 수집 -->
<script>
(function(){
  var hs = document.querySelectorAll('section.card h2');
  var list = document.getElementById('toc-list');
  if(!hs.length || !list) return;
  hs.forEach(function(h, i){
    if(!h.id) h.id = 'toc-' + i;
    var li = document.createElement('li');
    var a = document.createElement('a');
    a.href = '#' + h.id;
    a.textContent = h.textContent;
    li.appendChild(a); list.appendChild(li);
  });
  var links = list.querySelectorAll('a');
  var io = new IntersectionObserver(function(entries){
    entries.forEach(function(e){
      if(e.isIntersecting){
        links.forEach(function(l){ l.classList.remove('active'); });
        var a = list.querySelector('a[href="#' + e.target.id + '"]');
        if(a) a.classList.add('active');
      }
    });
  }, {rootMargin:'-8% 0px -80% 0px'});
  hs.forEach(function(h){ io.observe(h); });
})();
</script>

--c2#4a78b82편 (속도·가속도)c-blue

새 에피소드를 추가할 때는 assets/styles.css :root에 --c4 등을 추가하고 .c-NEW .step-no{ background:var(--c4); } 규칙도 함께 추가한다.

CTA 박스 배경 그라데이션 tint 예시:

  • --c1(빨강): #fdf0ef
  • --c2(파랑): #eef3fb
  • --c3(보라): #f3eff9
  • --green: #eef6ee

4. 체크리스트

신규 페이지 PR 올리기 전 확인 사항:

  • ☐ <head>에 3개 폰트 (Jua / Gowun Dodum / Gaegu 700) 모두 로드
  • ☐ <link rel="stylesheet" href="assets/styles.css"> 포함
  • ☐ 고정 .home-link (좌상단) + .home-link-bottom (본문 끝) 모두 있음
  • ☐ 모든 <section class="card">에 <h2> 있음 (사이드바 목차 자동 수집용)
  • ☐ .cta 섹션이 .home-link-bottom 바로 위에 위치
  • ☐ <nav class="toc-sidebar"> + TOC 스크립트가 </body> 직전에 있음
  • ☐ <footer> 문구가 © 2026 김경환 · CC BY-NC-SA 4.0 형식과 일치
  • ☐ docs/index.html TOC 그리드에 신규 페이지 링크 추가
  • ☐ 이전 편의 CTA 버튼이 신규 페이지를 가리키도록 수정

5. index.html 등록 방법

docs/index.html의 .toc-grid 안에 카드를 추가한다. 기존 카드를 복사해 수정하면 된다.

<a class="toc-card c-red" href="entry-game-N-slug.html">
  <span class="ep">N편</span>
  <span class="ttl">제목</span>
  <span class="desc">한 줄 설명</span>
</a>

학습 흐름 변경이 있으면 path strip도 함께 업데이트한다.

6. 주의 사항

  • 페이지별 <style>은 <head> 안에만 작성 (인라인 style은 사용하지 않음).
  • 페이지 공통 스타일을 바꾸고 싶으면 assets/styles.css를 수정하되, 모든 페이지에 영향이 가므로 주의.
  • Entry 블록 시각화(.hat, .stack, .cblk 등)의 색상은 실제 엔트리 에디터 색상과 일치해야 한다. 절대 탈색하지 말 것.
  • 캔버스 데모가 있는 페이지는 기존 <script> 뒤에 TOC 스크립트를 붙인다 (id="toc-list" 중복 없이).
  • dev-docs/ 파일은 배포되지 않으므로 여기에 설계 문서·메모를 자유롭게 써도 된다.
