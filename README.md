# 엔트리로 만드는 나만의 게임 🎮

엔트리(Entry)로 게임 캐릭터를 한 단계씩 만들어보는 **초등·중학생 대상 코딩 교육 자료**입니다.

👉 **[바로 보기](https://kkh029.github.io/entry-guide/docs/)**

---

## 목적

- 블록 코딩(엔트리)으로 **게임다운 움직임**을 직접 구현해보는 경험 제공
- 이동 → 속도 → 관성으로 이어지는 **물리 개념을 게임으로 자연스럽게 학습**
- 학교·학원·가정에서 바로 사용할 수 있는 무료 교육 자료

---

## 시리즈 구성

| 편 | 제목 | 핵심 개념 |
|---|---|---|
| 1편 | 내 캐릭터를 움직이게 만들자! | 화살표 이동, 충돌(바운딩 박스), 중력·점프 |
| 2편 | 이동·속도·가속도 — 게임 속 움직임의 비밀 | 속도 변수, 가속도, 스르륵 가속 |
| 3편 | 관성과 마찰 — 진짜 같은 움직임 | 관성, 마찰 계수, 서브스텝, 코요테 타임 |

> 1편 → 2편 → 3편 순서로 학습하면 자연스럽게 연결됩니다.
